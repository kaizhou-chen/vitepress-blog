# 配置别名 @

## 安装 @types/node

```bash
npm install @types/node --save-dev
```

## vite.config.ts 配置别名

```js
// 配置别名 @
resolve: {
    alias: {
        '@': path.resolve('./src')
    }
}
```

<img src="../../../assets/images/Vue3/vite.config.ts_alias.png" style="max-width: 700px;" />

## tsconfig.json 配置别名

```js
// 配置别名 @
"baseUrl": "./",
"paths": {
  "@/*": ["src/*"]
}
```
