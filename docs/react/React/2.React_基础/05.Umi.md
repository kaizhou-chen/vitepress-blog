# Umi



## 创建项目

PNPM

```bash
pnpm dlx create-umi@latest
```



### 配置端口 .env

**.env**

```
PORT=8888
```



### 配置代理 .umirc.ts

**.umirc.ts**

```js
import { defineConfig } from "umi";

export default defineConfig({
  routes: [
    { path: "/", component: "index" },
    { path: "/docs", component: "docs" },
  ],
  npmClient: 'yarn',
  
  // 配置代理
  proxy:{
    '/api': {
      target: 'http://my-test',
      changeOrigin: true,
      secure: false,
      pathRewrite: {
        '^/api': '/api',
      },
    },
  }
});
```
